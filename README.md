# Hello React Frontend

> A simple React front-end.

## Built With

- Javascript
- React, Redux
- Postgres

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Nodejs, NPM

### Setup

- Run `git clone https://github.com/mistermaxis/hello-react-front-end.git`
- Run `cd hello-react-front-end`
- Run `npm i`

### Usage

- Run `npm start`
- In your web browser, navigate to: `localhost:3000`
